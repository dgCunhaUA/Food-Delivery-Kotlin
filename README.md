## Setup backend

* docker run --name cm-proj1-mysql -p 3306:3306 -e MYSQL_ROOT_PASSWORD=root -d mysql
* mysql -u root
* mysql> CREATE DATABASE proj1;
* mysql> exit();
* npm install
* node models/client.model.js
* ...
